# Tic_Tac_Toe
Tic-Tac-Toe Game

This is a browser-based Tic-Tac-Toe game built using HTML, CSS, and JavaScript. It allows two players to play against each other on the same device, with a clean and interactive interface.

Features

Two-player gameplay: Player 1 uses X and Player 2 uses O.

Dynamic animations: Each move is animated when a player clicks a box.

Move counter: Tracks moves and automatically detects a draw if all 9 boxes are filled without a winner.

Winner announcement: Displays a congratulatory message highlighting the winning player.

Draw detection: Shows a “Match Draw!” message when no player wins.

Reset & New Game buttons: Reset the board and start a new game without reloading the page.

Stylish interface: Uses color-coded X and O, text animations, and shadow effects for a modern look.

Responsive design: Works on different screen sizes and devices.

Footer with credits: Maintains copyright and author info.

How to Play

Open the game in a web browser.

Player 1 clicks a box to place an X.

Player 2 clicks a box to place an O.

The first player to get three in a row (horizontally, vertically, or diagonally) wins.

If all 9 boxes are filled and no player has won, the game is a draw.

Click Reset or New Game to play again.
